# AI Workflow Engine Frontend

A modern React-based dashboard for the AI Workflow Orchestration Platform, built for executive demonstrations and production use.

## =€ Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Backend API server running on http://localhost:8080

### Installation & Setup

```bash
# Install dependencies
npm install

# Create environment file
cp .env.example .env

# Start development server
npm run dev

# Build for production
npm run build
```

The application will be available at `http://localhost:5173`
